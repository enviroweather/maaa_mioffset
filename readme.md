MI Offset is a critical regulatory tool used in siting livestock facilities. 
It uses a model developed by J. Andresen and M. T. Keifer in the department of
Geography and the Michigan Agicultural Climate office of Michigan State University. 

This project is the modernizing of the web interface to work with the 
visualization model code written by MT Keifer in Python.  

This is early stages of development and little is here.
